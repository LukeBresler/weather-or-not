# weather-or-not

Wherever you are, you can get the weather and a 5-day forecast.
This app is perfect for people who want to do things, and at a glance, you can!

Is it sunny? Maybe go to the beach.<br>
Is it partly cloudy? Why not go for a walk in the forest?<br>
Is it rainy? If so, you could skip camping and instead have a movie marathon with your buds.<br>

<h2>Accessibility:</h2>
The app allows vision-impaired users to listen to the current weather and a detailed 5-day forecast, ensuring everyone can stay up to date on the latest conditions.

<H2>Under the Hood</H2>
Leveraging <a href="https://developer.apple.com/documentation/corelocation" target= "_blank">Core Location</a> to get your location and the <a href="https://openweathermap.org/" target= "_blank">Open Weather Map API</a> to get the current weather. 

Once the data is accessed via the API, the code has been structured for modularity and ease of testing by using protocol-orientated programing (POP) principles.
